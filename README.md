## ▶️ Como Executar o Projeto

1. Crie o arquivo do projeto:
dotnet new blazorwasm -n MeuCurriculo

2. Acesse o arquivo do projeto:
cd MeuCurriculo

4. Execute a aplicação:
dotnet run

5. Abra no navegador:
[http://localhost:5207/]
